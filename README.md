# ESP32-SDcard-WebServer
项目名称：ESP32便携SD卡服务器

作者：B站 哈哈233jpg

项目实现文件管理、开机自连WIFI、设置（含OTA,域名前缀修改）、网页flash游戏和html游戏、播放视频、模式转换（AP STA AP+STA互相转换）、剪切板功能

成品尺寸:4\*3(3.1)\*1.18cm。
